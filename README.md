# 🏫 School Management API

A **Node.js + Express + MySQL REST API** that allows users to add schools and retrieve a list of schools sorted by proximity based on geographical location.

This project demonstrates backend development skills including API design, database integration, validation, and geolocation-based sorting.

---

## 🚀 Features

- ➕ Add new schools with name, address, latitude, and longitude
- 📍 Get list of schools sorted by distance from user location
- 🧠 Input validation using Joi
- ⚡ Clean MVC architecture
- 🗄️ MySQL database integration
- ❌ Global error handling & 404 middleware
- 📦 Modular and scalable backend structure

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MySQL
- Joi (Validation)
- dotenv
- mysql2
